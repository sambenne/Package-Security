## [0.1.0] - TBD

### Added

- Laravel `package:audit` artisan command.
- Standalone `vendor/bin/package-security` binary.
- Composer vulnerability, abandoned package, outdated package, freshness, and licence checks.
- npm vulnerability, outdated package, freshness, and licence checks.
- Configurable vulnerability fail thresholds.
- Configurable update reporting.
- Configurable release freshness quarantine.
- Configurable licence allow/block policy.
- Table, JSON, and SARIF report output.
- CI exit codes.
- GitHub Actions test matrix for Laravel 10, 11, and 12.
- GitHub Actions integration documentation.
